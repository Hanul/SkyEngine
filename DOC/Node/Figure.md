# 도형

## 직선 노드
직선을 만듭니다.

```javascript
let line = SkyEngine.Line({
	y : -140,
	startX : -100,
	startY : -10,
	endX : 100,
	endY : 10,
	border : '5px solid red'
}).appendTo(SkyEngine.Screen);
```

[노드](../Node.md)의 파라미터들 외에 사용 가능한 파라미터는 다음과 같습니다.
- `startX`
- `startY`
- `endX`
- `endY`
- `border`
